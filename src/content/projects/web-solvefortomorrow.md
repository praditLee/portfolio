เริ่มจากการแก้ปัญหาเว็บไซต์โดน DDoS Attack ด้วยการปรับเว็บไซต์จากเดิมที่สร้างด้วย WordPress ไปสู่ Astro อัพเดทผ่าน github แล้ว Deploy บน Cloudflare แสดงผลรวดเร็วและแก้ปัญหาการถูกโจมตีเพราะโฮสต์ไม่แข็งแรง แสดงผลได้ทุกขนาดหน้าจอตามมาตราฐาน Tailwind CSS มีระบบ Google Login เพื่อจัดการการเขัาถึงของ User

จากนั้น พัฒนาระบบต่อให้เชื่อมกับฐานข้อมูล SupaBase เพื่อทำระบบบริหารจัดการ user ให้เข้าร่วมการแข่งขันในรอบต่างๆ ที่มีระบบการเรียนออนไลน์ การส่งงาน การส่งโครงงานประกวด และการนัดหมายเข้าร่วมกิจกรรมต่างๆ ในแต่ละรอบ
